### Description

Write a function that calculates x raised to the power of n (xⁿ) recursively.  Handle both positive and negative exponents.

### Constraints

- -100 ≤ x ≤ 100
- -10 ≤ n ≤ 10
- x != 0 when n < 0

### Example

Input: x = 2, n = 3
Output: 8

Input: x = 5, n = -2
Output: 0.04

### Concepts Covered

- Recursion
- Handling negative exponents
- Floating-point arithmetic
