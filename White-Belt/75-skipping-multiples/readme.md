### Description
Given an integer `limit` and an integer `skip_value`, print all numbers from 1 to `limit` (inclusive), skipping the multiples of `skip_value`.

### Constraints
*   `1 <= limit <= 100`
*   `1 <= skip_value <= 10`

### Example
If `limit` is 10 and `skip_value` is 3, the output should be:
1 2 4 5 7 8 10

### Concepts Covered
*   `for` loop
*   `if` statement
*   `continue` statement
