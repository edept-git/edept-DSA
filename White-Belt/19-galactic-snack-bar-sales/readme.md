### Description

The Galactic Snack Bar is tracking its daily sales.  They sell three types of snacks: Fizzbangs, ZapZaps, and Glimmerglobs.  Given the number of each snack sold, calculate the total number of snacks sold.

### Constraints

* The number of each snack type sold will be a non-negative integer.

### Example

Input: Fizzbangs = 10, ZapZaps = 5, Glimmerglobs = 20
Output: 35

### Concepts Covered

* Variables
* Integer Data Types
* Arithmetic Operations
