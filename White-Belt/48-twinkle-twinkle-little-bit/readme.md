### Description

Given an integer `n`, determine if it's a power of 2.  A power of 2 is a number of the form 2<sup>k</sup> where k is a non-negative integer.

### Constraints

* 0 <= n <= 10<sup>9</sup>

### Example

Input: 16
Output: true

Input: 10
Output: false

### Concepts Covered

* Bitwise AND operator
* Properties of powers of 2
